# LAB NOC 01 – Console Access & Password Recovery

## 🚀 Summary
Real NOC scenario using a Cisco Catalyst 2960 switch where console access was required after loss of remote connectivity.

## 🧩 Problem
Remote access (SSH/Telnet) was unavailable, requiring physical console access.

## ⚙️ Solution
- USB–RJ45 console connection  
- COM port identification (COM5)  
- PuTTY configuration (9600, 8N1)  
- Boot interruption (MODE button)  
- Config recovery (config.text, vlan.dat)  

## ✅ Result
Switch restored to factory-clean configuration without service disruption.
